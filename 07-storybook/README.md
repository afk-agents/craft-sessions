# Craft Session #7: Storybook

Create an illustrated children's picture book with Claude.

## Get Started

Download the `CLAUDE.md` file into your project directory:

```bash
curl -O https://raw.githubusercontent.com/afk-agents/craft-sessions/main/07-storybook/CLAUDE.md
```

Then start a conversation with Claude in that directory and tell it the story you want to bring to life.
